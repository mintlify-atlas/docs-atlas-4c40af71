## Endpoint

`DELETE /api/customers/delete`

Deletes a customer from Stripe. The customer ID can be provided either in the request body as `userId` or as a URL parameter.

## Request Body

<ParamField body="userId" type="string" required>
  The Stripe customer ID to delete (format: `cus_...`)
</ParamField>

<Info>
Alternatively, you can pass the customer ID as a URL parameter (`/api/customers/:id`) instead of in the request body.
</Info>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  Object containing the ID of the deleted customer
  
  <Expandable title="data properties">
    <ResponseField name="id" type="string">
      The Stripe customer ID that was deleted
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X DELETE https://your-domain.com/api/customers/delete \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "cus_1234567890abcdef"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-domain.com/api/customers/delete', {
  method: 'DELETE',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    userId: 'cus_1234567890abcdef'
  })
});

const data = await response.json();
console.log(data);
```
</CodeGroup>

## Response Examples

### Success (200)
```json
{
  "status": true,
  "message": "Usuario eliminado con éxito",
  "data": {

}
```
