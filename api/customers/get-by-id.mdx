---

## Endpoint

`GET /api/customers/:id`

Retrieves detailed information about a specific customer using their Stripe customer ID.

## Request

<ParamField path="id" type="string" required>
  The Stripe customer ID (format: `cus_...`)
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Response message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  Complete Stripe customer object with all customer details including:
  
  <Expandable title="data properties">
    <ResponseField name="id" type="string">
      Unique Stripe customer ID
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Customer's email address
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Customer's full name
    </ResponseField>
    
    <ResponseField name="phone" type="string">
      Customer's phone number
    </ResponseField>
    
    <ResponseField name="created" type="integer">
      Timestamp when the customer was created
    </ResponseField>
    
    <ResponseField name="default_source" type="string">
      ID of the customer's default payment source
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X GET https://your-domain.com/api/customers/cus_1234567890abcdef \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const customerId = 'cus_1234567890abcdef';

const response = await fetch(`https://your-domain.com/api/customers/${customerId}`, {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```
</CodeGroup>
